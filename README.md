# 🎯 Jogo do Número Secreto

Aplicação interativa desenvolvida com JavaScript onde o usuário deve adivinhar um número aleatório dentro de um intervalo definido.

## 🚀 Objetivo

Praticar lógica de programação, manipulação do DOM e criação de interatividade em aplicações web.

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

---

## 🎮 Como funciona

- O sistema gera automaticamente um número secreto entre 1 e 10
- O usuário deve tentar adivinhar o número
- A cada tentativa, o sistema informa se o número é maior ou menor
- O jogo impede:
  - valores inválidos
  - números repetidos
- O número de tentativas é contabilizado
- Ao acertar, o jogo exibe:
  - quantidade de tentativas
  - histórico de números digitados
- É possível reiniciar o jogo a qualquer momento

---

## 📌 Funcionalidades

- ✅ Validação de entrada
- ✅ Controle de tentativas
- ✅ Feedback dinâmico ao usuário
- ✅ Histórico de chutes
- ✅ Botão de reinício
- ✅ Interface estilizada com CSS
